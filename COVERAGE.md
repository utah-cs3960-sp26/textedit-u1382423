# Coverage Guide
## How to Collect Coverage for the Text Editor
Set the environment:
```bash
source setenv.sh
```

Running the tests:
```bash
pytest testing/test_text_editor.py -v
```

Printing the Coverage metrics:
```bash
python testing/run_coverage.py
```
